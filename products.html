Plum iTV Transcoder

Plum iTV transcoder is a fast, scalable and highly flexible software based video encoding system supporting HEVC, MPEG-4 AVC and MPEG-2 video formats. Due to highly optimized software architecture it uses full CPU capacity, thus transcoding up to several hundred video streams simultaneously on standard high performance server.

Plum iTV Media Engine

Streamer, Catchup server, Storage, VoD processing server

Plum iTV media streamer delivers standard OTT content formats (HLS) to the end-users. It also has Catchup features incorporated enabling up to 7-day catchup functionality for any channel. It includes VOD ingest module that prepares assets for the system.

Plum iTV Middleware

Core, Provisioning, Billing engine, Statistics and Analytics engine

Plum iTV Middleware provides everything necessary to run a modern multimedia application for end-users. As a central software component, Plum iTV Middleware controls all other sub-systems (VOD, Catchup, LiveTV) and connects end-user clients.
